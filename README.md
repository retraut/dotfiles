### What is this repository for? ###
General dotfiles for my server

### Features ###
* Installs or updates you doftiles configuration 
* language - bash
* version - 1.1b

### How do I get set up? ###
* Clone repo with 

```
#!shell
git clone git@bitbucket.org:Retraut/dotfiles.git ~/.dotfiles

```

* Run **~/.dotfiles/dots install**
